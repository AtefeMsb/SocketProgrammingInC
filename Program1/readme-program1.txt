Programming Assignment 1:
Client using customized protocol on top of UDP protocol for sending information to the server.

Compiling instructions:

Open 2 terminals, one for client and one for the server.

Step1: Compile program1server.c file by command:
  gcc -o server program1server.c

Step2: Compile program1client.c file by command:
  gcc -o client program1client.c

Step3: Execute the server by command:
	./server 

Step4: Execute the client by command:
	./client

Programming Assignment 2:
Client using customized protocol on top of UDP protocol for requesting identification from server for access permission to the network.

Compiling instructions:

Open 2 terminals, one for client and one for the server.

Step1: Compile program2server.c file by command:
  gcc -o server program2server.c

Step2: Compile program2client.c file by command:
  gcc -o client program2client.c

Step3: Execute the server by command:
	./server 

Step4: Execute the client by command:
	./client
